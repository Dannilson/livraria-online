# Livraria Online - Backend

Projeto Spring Boot para gestão de livros e escritores, desenvolvido como parte da avaliação A2 de Backend Frameworks.

## Tecnologias Utilizadas

- Java 17
- Spring Boot 3.2.0
- Spring Data JPA
- H2 Database (para desenvolvimento)
- Lombok
- ModelMapper
- Swagger (OpenAPI 3.0)
